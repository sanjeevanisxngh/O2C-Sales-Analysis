# Sales Data Analysis using Order-to-Cash (O2C)

## Project Overview
This project is based on the SAP Order-to-Cash process and focuses on sales data analysis using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- CSV Dataset
- VS Code

## Analysis Done
- Total Sales
- Product-wise Sales
- Region-wise Sales
- Payment Status Analysis
- Monthly Sales Trend

## Project Output
The project shows important business insights using charts and data analysis.
